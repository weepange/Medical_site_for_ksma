# Клиника «Рождение» — статический сайт

Одностраничный адаптивный сайт частной клиники с фокусом на ведение беременности, роды и послеродовое сопровождение.

## Что реализовано

- Главный экран с CTA-кнопками и навигацией по секциям.
- Разделы: о клинике, услуги, специалисты, отзывы, контакты.
- Модальные окна для карточек врачей, подробностей услуг, полных отзывов и формы записи.
- Интерактивная форма записи с выбором услуги и цены из dropdown.
- Проверка ФИО в форме записи (только корректный текстовый ввод).
- Маска и проверка телефона в формате `+7 (XXX) XXX-XX-XX`.
- Проверка даты записи (дата не может быть раньше текущей).
- Плавные анимации появления блоков (`IntersectionObserver`).
- Горизонтальная прокрутка карточек команды и отзывов (в том числе колесом мыши на десктопе).
- Адаптив для планшетов и мобильных устройств.

## Стек

- `HTML5`
- `CSS3` (переменные, адаптивные media queries)
- `Vanilla JavaScript` (без фреймворков и сборки)

## Запуск локально

Проект не требует установки зависимостей.

1. Перейдите в директорию проекта:

```bash
cd /home/wpng1337/project_kubmsu
```

2. Запустите локальный сервер:

```bash
python3 -m http.server 8080 --directory site
```

3. Откройте в браузере:

```text
http://localhost:8080
```

## Где менять контент

- Тексты, названия услуг, цены, карточки врачей и отзывы: `site/index.html`.
- Цвета, типографика, размеры, адаптив и анимации: `site/styles.css`.
- Логика модальных окон, dropdown, валидация формы и интерактив: `site/script.js`.
- Изображения: `site/images/**`.

## Важно при доработке

- При добавлении новой услуги обновляйте карточку, модальное окно и список опций в форме записи (`#booking-dropdown-menu`).
- Если меняете телефонную валидацию, синхронизируйте функции `formatPhone` и `validatePhone`.
- Если переносите проект на продакшен, добавьте обработчик формы на сервере (сейчас отправка имитируется на клиенте).
